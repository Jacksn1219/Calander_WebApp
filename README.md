# Necessary Commands:

## Check Directory 

```
cd src
```

## Frontend:

- Correct directory:
```
cd client
```

- For running:
```
yarn start
```

- For restarting (after every change made):
```
yarn restart
```
- For watch-mode:
```
yarn watch
```

## Backend:

- Correct directory:
```
cd server
```

- For running:
```
yarn start
```

- For restarting (after every change made):
```
yarn restart
```
- For watch-mode:
```
yarn watch
```
