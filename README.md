# Necessary Commands:

## Check Directory 

```
cd src
```

## Frontend:

- Correct directory:
```
cd client
```

- For running:
```
yarn start
```
- For building:
```
yarn build
```
- For testing:
```
yarn test
```

##
If necessary you can substitute the 'yarn' for 'npm'
